## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/manuj-tiwari2906/Viriciti-be-assignment.git

2. **Navigate to Repository:**

    cd your-repository-name

3. **Install dependencies:**

    npm install

4. **Start Project:**

  npm start

5. **Run Tests:**

  npm test

6. **Build application**

  npm run build
